## Closed Source

The following Tyk components, created and maintained by the Tyk Team, are proprietary and closed-source:

* [Tyk Dashboard](/nightly/tyk-dashboard)
* [Tyk Developer Portal](/nightly/tyk-developer-portal)
* [Tyk Multi Data Center Bridge](/nightly/api-management/mdcb#managing-geographically-distributed-gateways-to-minimize-latency-and-protect-data-sovereignty)
* [Universal Data Graph](/nightly/api-management/data-graph#overview)
* [Tyk Operator](/nightly/api-management/automations/operator#what-is-tyk-operator)
* [Tyk Sync](/nightly/api-management/automations/sync)

If you plan to deploy and use the above components On-premise, license keys are required.
