## Closed Source

The following Tyk components, created and maintained by the Tyk Team, are proprietary and closed-source:

* [Tyk Dashboard](/docsv2/5.7/tyk-dashboard)
* [Tyk Developer Portal](/docsv2/5.7/tyk-developer-portal)
* [Tyk Multi Data Center Bridge](/docsv2/5.7/api-management/mdcb#managing-geographically-distributed-gateways-to-minimize-latency-and-protect-data-sovereignty)
* [Universal Data Graph](/docsv2/5.7/api-management/data-graph#overview)
* [Tyk Operator](/docsv2/5.7/api-management/automations/operator#what-is-tyk-operator)
* [Tyk Sync](/docsv2/5.7/api-management/automations/sync)

If you plan to deploy and use the above components On-premise, license keys are required.
