The Tyk Pump component takes all of the analytics in Tyk and moves the data from the Gateway into your Dashboard. It is possible to set it up to send the analytics data it finds to other data stores. Currently we support the following:

- MongoDB or SQL (Used by the Tyk Dashboard)
- [CSV](/5.6/product-stack/tyk-pump/advanced-configurations/configure-data-sinks/csv)
- [Elasticsearch (2.0 - 7.x)](/5.6/product-stack/tyk-pump/advanced-configurations/configure-data-sinks/elasticsearch)
- Graylog
- Resurface.io
- InfluxDB
- [Moesif](/5.6/tyk-configuration-reference/tyk-pump-configuration/moesif)
- [Splunk](/5.6/tyk-configuration-reference/tyk-pump-configuration/splunk)
- StatsD
- DogStatsD
- Hybrid (Tyk RPC)
- [Prometheus](/5.6/tyk-stack/tyk-pump/other-data-stores/monitor-apis-prometheus)
- [Logz.io](/5.6/product-stack/tyk-pump/advanced-configurations/configure-data-sinks/logzio)
- Kafka
- Syslog (FluentD)

See the [Tyk Pump Configuration](/5.6/tyk-pump) for more details.
