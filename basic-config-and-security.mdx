This section covers methods for configuring and applying security methods to Tyk. the following subjects are covered:

- [Security](/api-management/security-best-practices#securing-apis-with-tyk). This section includes how to apply security to your Tyk components, your APIs, and security policies.
- [Reduce latency](/api-management/gateway-optimizations#). This section describes methods for reducing latency in your Tyk installation.
- [Log traffic](/api-management/logs-metrics#logging-api-traffic). This section describes how to control the capture of traffic logs for the transactions (API requests and responses) handled by your Tyk Gateways.
- [Gateway events](/api-management/gateway-events#event-categories). This section describes the different types of asynchronous events that Tyk Gateway can generate and how you can register webhooks and other methods to handle them.
