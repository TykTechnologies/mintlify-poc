The SecurityPolicy custom resource defines configuration of [Tyk Security Policy object](/basic-config-and-security/security/security-policies).
Here are the supported features:

| Feature                                                   | Supported                                                      |
| :----------------------------------------------------------- | :---------------------------------------------------------------- |
| API Access                     | ✅ |
| Rate Limit, Throttling, Quotas | ✅ |
| Meta Data & Tags           | ✅ |
| Path based permissions     | ✅ |
| Partitions                 | ✅ |
| Per API limit              | ❌ |

Consult [Security Policy CRD](/tyk-stack/tyk-operator/secure-an-api#securitypolicy-crd) to see examples of setting these features.
